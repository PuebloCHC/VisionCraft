FAQ

Does this use AI?

The generator uses heuristic templates that incorporate your prompt. You can optionally paste an API key to expand prompts locally (coming soon). Everything runs client‑side.

Is my idea sent to a server?

No. All code is produced in your browser and downloaded as a ZIP. There’s no backend.

Can I deploy this site on GitHub Pages?

Yes. Place this index.html in your repo root or /docs, then enable Pages. That’s it.
